# Development & Testing Scripts

This directory contains scripts for development and testing, separate from
the EMNLP paper experiments in `../experiments/`.

## Scripts

- `benchmark.py` - Benchmark runner for ablation studies
- `live_demo.py` - Live Anthropic API demo for testing

## vs experiments/

The `experiments/` directory contains scripts specifically for the EMNLP
submission (run_experiments.py, analyze_experiments.py, etc.). This directory
is for general development and testing.
